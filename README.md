# 🎬 Netflix Clone

A responsive **Netflix Clone** built using **HTML5 and CSS3**, inspired by Netflix's official landing page. The project focuses on recreating the UI with a clean, modern, and responsive design.

## 🌐 Live Demo

🔗 https://pranav10-dotcom.github.io/Netflix_Clone/

---

## 📂 Project Structure

```
Netflix_Clone/
│── assets/          # Images and icons
│── videos/          # Background videos
│── index.html       # Main webpage
│── style.css        # Styling
└── README.md
```

---

## ✨ Features

- 🎥 Netflix-inspired landing page
- 📱 Fully responsive design
- 🎨 Clean and modern UI
- 🎬 Video background support
- 🖼️ High-quality images and assets
- ⚡ Smooth hover effects
- 🌍 Cross-browser compatibility

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/Pranav10-dotcom/Netflix_Clone.git
```

### Open the project

```bash
cd Netflix_Clone
```

Open **index.html** in your preferred browser.
